## Photoshop 

### 抠图

```
魔棒工具
	图片背景 白色 或 纯色
	选择 背景，按住 Delete ，背景被删除
	属性：连续，容差

快速选择工具

套索工具
	去掉多余的选区 按住 ALT 框住选区线

橡皮擦工具
	背景橡皮擦工具
		取样一次 背景颜色
		自动擦除背景
		(注：按下鼠标取样，擦除时不要松开鼠标)
	魔术棒橡皮擦工具

钢笔工具
	(注：最好勾选 像皮带)
	点击 建立锚点 主线路径
	点击 + 拖动 建立锚点 弧线路径
	按住 CTRL 调整两个手柄
	按住 ALT 调整单个手柄
	按住 ALT + 点击锚点 取消单个手柄(之后又是直线)
	右键菜单 建立选区 添加蒙版

通道
	点击通道面板
	新增通道
	图像 --> 应用图像 --> 混合：颜色加深 --> 主体部分颜色 更黑
	色阶(CTRL + L) 黑色区域更黑、白色区域更白
	涂黑色：画笔工具 前景色(黑色) 按住鼠标 将需要不透明的主体部分 涂为 黑色
	减淡白色：减淡工具 范围(高光)
	按住 CTRL 点击通道缩略图
	CTRL + SHIFT + I 反选选区
	选中 RGB 通道
	点击 图层面板 
	添加蒙版

选框工具
	椭圆形选框工具 
		视图 --> 标尺 建立参考线
		按住 SHIFT + 拖动鼠标 建立正圆 
		添加图层蒙面

	矩形选框工具 
		按住并拖动鼠标 建立选区
		ALT + 鼠标 继续 建立选区
```
